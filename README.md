#作用就是让公共dns作为跳板。公告dns速度快，而且节点多。
可以大大增加网站的解析速度。可以阻止中共的防火墙阻断dns。
 在内地搭建一个服务器。在内地开放53端口 是域名解析找到这个ip上。 
这个服务器上连接VPN 直接连接境外网络。然后让公共dns缓存。
可以破解防火墙阻断域名权威服务器被阻断。
